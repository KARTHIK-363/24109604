# Investigating the Impact of Noise and Outliers on Support Vector Machine Decision Boundaries

##  Overview
This project investigates how noise and outliers affect the behaviour of Support Vector Machine (SVM) classifiers. The study focuses on analysing changes in decision boundaries, support vectors, and classification performance under different data conditions.

---

##  Objectives
- Analyse SVM performance on clean data  
- Study the effect of feature noise  
- Investigate the impact of outliers  
- Understand how support vectors influence the decision boundary  
- Evaluate the role of the regularisation parameter (C)  

---

## Experiments Conducted
The following scenarios were analysed:

1. Clean Dataset  
2. Dataset with Feature Noise  
3. Dataset with Outliers  
4. Effect of Regularisation Parameter (C)  

---

##  Key Findings
- SVM performs well on clean data  
- Noise reduces accuracy and stability  
- Outliers significantly distort decision boundaries  
- Support vectors are highly sensitive to extreme points  
- Regularisation parameter (C) controls robustness
